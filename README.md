# Connect 4 Android Game (Group Project)

## 📱 Description
An Android-based Connect 4 game developed as part of a Mobile Application Development project. The application supports multiple gameplay modes, customization options, and interactive UI features, demonstrating core concepts of Android development, game logic, and user interface design.

## 🚀 Features
- Player vs Player mode
- Player vs AI mode (random move logic)
- Adjustable grid sizes
- Custom player names and disc colours
- Avatar-based player profiles
- Gameplay statistics tracking
- Undo and reset functionality
- In-game notifications and turn indicators

## 🛠️ Technologies Used
- Java
- Android Studio
- Android SDK
- XML (UI Design)
- SharedPreferences

## ▶️ How to Run
1. Open the project in Android Studio
2. Let Gradle sync complete
3. Run the app on an emulator or physical device

## 📚 Project Context
This project was developed as part of a university assignment focusing on Android application development, including UI design, game logic, modular architecture, and state management.
